## 概覽

**C08 Minecraft Server** 融合科技美學與遊戲深度，標誌靈感源於現代 AI 的幾何結構，象徵網絡、宇宙與玩家之間的連結。我們提供穩定且充滿創意的環境，讓你在開放世界中建築、探險，與全球玩家共同協作。

---

## 伺服器特色

- **極簡科技風格**：專屬現代幾何標誌，為伺服器注入未來感。
- **開放世界生存**：豐富的地形生成與平衡的生存機制。
- **創意建築區**：提供無限資源，讓想像力不受限制。
- **高效能核心**：採用 Spigot / Paper 端，確保流暢無延遲。
- **社群互動**：整合 Discord 同步聊天，隨時與隊友保持聯繫。

## 快速加入

### 伺服器資訊

- **IP 位址**：`26.77.136.232`
- **遊戲版本**：Minecraft Java Edition 26.2
- **正版驗證**：開啟 (Online Mode)

### 加入步驟

1. 啟動 Minecraft 啟動器。
2. 前往 **多人遊戲** → **新增伺服器**。
3. 輸入名稱 `C08 Server` 及上述 IP。
4. 點擊 **完成**，開始冒險。
